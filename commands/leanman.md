# /leanman [lite|full|ultra|wenyan-lite|wenyan-full|wenyan-ultra|off]

Set leanman intensity level. Default: full. No argument reports current level.

- **lite**: prefer simple, name lazier alternative. Prose: no filler/hedging, keep articles + full sentences.
- **full** (default): ladder enforced + anti-slop prose + caveman compression (drop articles, fragments OK).
- **ultra**: YAGNI extremist, challenge requirements. Prose: bare fragments, one word when enough.
- **wenyan-lite**: semi-classical Chinese. Drop filler/hedging, keep grammar structure.
- **wenyan-full**: maximum classical Chinese (文言文). 80-90% character reduction.
- **wenyan-ultra**: extreme classical abbreviation.
- **off**: disable leanman.

Rules activated: code minimalism ladder (YAGNI: stdlib: native: one line) + anti-slop prose (no filler, active voice, no em-dash) + caveman compression (no self-reference, no invented abbreviations, no causal arrows, auto-clarity).
