# /leanman [lite|full|ultra|off]

Set leanman intensity level. Default: full. No argument reports current level.

- **lite**: prefer simple, name lazier alternative
- **full** (default): ladder enforced + anti-slop prose
- **ultra**: YAGNI extremist, challenge requirements
- **off**: disable leanman

Rules activated: code minimalism ladder (YAGNI → stdlib → native → one line) + anti-slop prose (no filler, active voice, no em-dash).
