# /leanman-help

Quick reference for leanman commands.

Commands:
- `/leanman [lite|full|ultra|wenyan-lite|wenyan-full|wenyan-ultra|off]` — set level
- `/leanman-review [--all]` — review over-engineering + slop + caveman compression
- `/leanman-help` — this card

Levels: lite (light prose), full (default: code ladder + anti-slop + caveman compression), ultra (extreme). Wenyan = classical Chinese.

Caveman: no self-reference, no invented abbreviations, no causal arrows, auto-clarity.

Deactivate: "stop leanman" / "normal mode".
